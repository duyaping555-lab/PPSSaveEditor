# PPSSPP 存档编辑器 (Android)

这是一个用于编辑 PPSSPP 模拟器存档文件的 Android 应用程序。你可以用它来修改游戏中的金钱、HP、物品数量等数值。

## 功能特性

- 📁 **浏览和选择存档**: 从手机存储中选择 PSP 存档文件夹
- 📋 **SFO 信息解析**: 显示游戏标题、存档标题等信息
- 🔓 **存档加解密**: 支持 PSP 存档的加密/解密（模式 1, 3, 5）
- 📝 **十六进制编辑器**: 可视化编辑存档数据的每个字节
- 🎮 **CWCheat 代码**: 支持使用标准的 CWCheat 补丁代码修改存档

## 使用说明

### 准备工作

1. 在安卓手机上安装 PPSSPP 模拟器
2. 确保你有想要修改的游戏存档

### 使用步骤

1. **导出存档**
   - 在 PPSSPP 中: 设置 → 工具 → 开发工具 → 保存数据备份
   - 或者直接从 `Android/data/org.ppsspp.ppsspp/files/PSP/SAVEDATA/` 复制存档文件夹

2. **打开存档编辑器**
   - 启动 PPSSaveEditor 应用
   - 授予存储权限

3. **选择存档**
   - 点击"选择存档文件夹"
   - 选择包含 `PARAM.SFO` 和 `DATA.BIN` 的存档文件夹

4. **解密存档**
   - 如果存档是加密的，点击"解密"按钮
   - 解密成功后才能编辑

5. **编辑存档**

   **方法一: 十六进制编辑器**
   - 点击"打开十六进制编辑器"
   - 找到要修改的数值位置
   - 点击字节进行修改
   - 支持搜索、跳转、修改不同位宽数值

   **方法二: CWCheat 代码**
   - 点击"应用补丁代码"
   - 输入 CWCheat 格式的代码，例如:
     ```
     _L 0x20555678 0x0098967F  ; 金钱 9999999
     _L 0x2055567C 0x0000270F  ; HP 9999
     ```
   - 点击"应用代码"

6. **加密并保存**
   - 修改完成后，点击"加密"按钮
   - 将加密后的文件保存回原始位置

7. **导入存档**
   - 将修改后的存档文件夹复制回 PPSSPP 的 SAVEDATA 目录
   - 在 PPSSPP 中加载存档

### CWCheat 代码格式

代码格式: `_L 0xTXXXXXXX 0xYYYYYYYY`

- `T` = 类型:
  - `0` = 8位写入 (0x0XXXXXXX 0x000000YY)
  - `1` = 16位写入 (0x1XXXXXXX 0x0000YYYY)
  - `2` = 32位写入 (0x2XXXXXXX 0xYYYYYYYY)
- `XXXXXXX` = 内存地址（相对于存档起始位置的偏移）
- `YYYYYYYY` = 要写入的值

### 查找修改地址的方法

1. **使用现有的 CWCheat 表**
   - 在网上搜索游戏的 CWCheat 代码
   - 将内存地址转换为存档偏移

2. **使用十六进制搜索**
   - 在修改前备份存档
   - 在游戏中改变数值（如花钱）
   - 比较两个存档的差异
   - 用十六进制编辑器搜索已知的数值

3. **使用 GameGuardian (需要 Root)**
   - 在 PPSSPP 运行时搜索内存
   - 找到地址后计算存档偏移

## 技术说明

### PSP 存档结构

```
SAVEDATA/
├── PARAM.SFO      # 存档元数据（未加密）
├── ICON0.PNG      # 存档图标
├── DATA.BIN       # 存档数据（加密）
└── SECURE.BIN     # 某些游戏的额外数据
```

### 加密说明

PSP 存档使用基于 AES 的加密，支持三种模式:
- **模式 1**: 使用固定密钥
- **模式 3**: 使用游戏特定密钥
- **模式 5**: 使用密钥并附加哈希验证

## 构建项目

### 要求

- Android Studio Hedgehog (2023.1.1) 或更高版本
- Android SDK 34
- Kotlin 1.9.0+

### 构建步骤

1. 克隆或下载项目
2. 在 Android Studio 中打开项目
3. 同步 Gradle 文件
4. 构建 APK: Build → Build Bundle(s) / APK(s) → Build APK(s)

## 注意事项

⚠️ **警告**
- 修改前务必备份原始存档
- 不正确的修改可能导致存档损坏
- 某些游戏有额外的校验，修改后可能无法加载
