使用说明
--------
1. Python命令行版本:
   python psp_save_editor.py <存档路径> --decrypt --hex

2. Kivy GUI版本:
   cd mobile_app
   python main.py

3. Android APK构建:
   cd mobile_app
   buildozer android debug

4. 运行测试:
   python test_editor.py

5. 运行演示:
   python demo.py
